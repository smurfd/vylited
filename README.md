# winterd

A Colorscheme for VIM highly based on 'vylight' by Vy-Shane Sin Fat 
https://github.com/vim-scripts/vylight

Works best in GVIM 

For it to work great in a Terminal, 
add 'set termguicolors' to your ~/.vimrc file
